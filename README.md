# مدير شبكتك (Street Net Manager)

Network management application by Al-Salam.

## Tech Stack

- **Frontend**: React 19, Vite
- **Desktop**: Electron
- **Data**: In-memory (no external database required)

## Development

```bash
npm install
npm run electron:dev
```

## Build

```bash
npm run electron:build
```

Output is in the `release/` folder.
